   \page Installation

   The expression library is built using <a
   href="http://www.cmake.org/">CMake</a>.  Be sure that you have a
   relatively recent version of CMake installed on your machine.

   \section radprops_depends Required Libraries:
   The following libraries are required to build the RadProps library:

   - <a href="http://www.boost.org"> Boost libraries</a>.
   
   - <a href="http://www.cmake.org"> CMake build system</a>
   
   - <a href="http://git-scm.com/">git version control system</a> (to check out RadProps source)
   
   - Cantera (for preprocessor only) 


   \section install_steps Build Instructions:

   -# Install the \link radprops_depends required libraries \endlink as
      listed above.
   -# Clone the TabProps library: \code git clone git://software.crsim.utah.edu/TabProps.git RadProps \endcode
   -# create a build directory, e.g. \code mkdir myBuildDir; cd myBuildDir \endcode
   
   \subsection prepromode Preprocessing mode
   Here is a suggested configuration for building TabProps in preprocessing mode:
   \code
    cmake [path to src] \
     -DTabProps_ENABLE_TESTING=ON \
     -DTabProps_PREPROCESSOR=ON \
     -DTabProps_UTILS=ON \
     -DCantera_CMAKE=ON \
     -DCantera_DIR=[cantera_install_dir/share] \
     -DCMAKE_BUILD_TYPE=Release
     -DCMAKE_INSTALL_PREFIX=[install_dir]
   \endcode
   
   \subsection prodmode Production (CFD) mode
   Here is a suggested configuration for building TabProps in a minimalist mode
   so that only what is needed to load tables into a downstream application code
   will be built:
   \code
    cmake [path to src] \
     -DCMAKE_INSTALL_PREFIX=[install_dir]
     -DTabProps_ENABLE_TESTING=OFF \
     -DTabProps_PREPROCESSOR=OFF \
     -DTabProps_UTILS=OFF \
     -DCMAKE_BUILD_TYPE=Release
   \endcode
     
*/
